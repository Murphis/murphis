## Installation

<Steps>
    <Step>
    Install shadcn/ui popover and command components

    ```bash
    npx shadcn@latest add popover command
    ```
    </Step>
    <Step>
    Install Murphy TokenIcon

      <InstallationCommands
      packageUrl={`${process.env.NEXT_PUBLIC_BASE_URL}/r/token-icon.json`}
    />
    </Step>
        <Step>
    Install Murphy Token Combobox

      <InstallationCommands
      packageUrl={`${process.env.NEXT_PUBLIC_BASE_URL}/r/token-combobox.json`}
    />
    </Step>
    <Step>
    Use TokenCombobox
</Step>

</Steps>

## Basic Usage
```tsx
import {TokenCombobox} from "@/components/ui/murphy/token-combobox.tsx"
```
```tsx
<TokenCombobox
assets={assets}
address={publicKey}
showBalances={!!publicKey}
onSearch={searchAssets}
/>

```
