## Installation

<Steps>
  <Step>
    Install dependencies

    Start by installing required Solana dependencies

    <DependenciesInstallCommands packageUrl={"@solana/web3.js @solana/spl-token @solana/wallet-adapter-react @lightprotocol/stateless.js @lightprotocol/compressed-token bs58 dotenv"}/>

  </Step>

  <Step>
    Add Wallet Provider

    Make sure you have added the Wallet Provider to your application. If not, follow the steps in the [Connect Wallet Button](/docs/onchainkit/connect-wallet-button) guide first.

  </Step>

  <Step>
    Add Claim Token Form

    <InstallationCommands
      packageUrl={`${process.env.NEXT_PUBLIC_BASE_URL}/r/claim-cToken.json`}
    />

  </Step>
</Steps>

### Basic Usage

```tsx
import { ClaimTokenForm } from "@/components/ui/murphy/claim-cToken";
```

```tsx
export default function MyPage() {

    </div>
  );
}
