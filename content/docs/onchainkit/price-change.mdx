## Basic Usage
    ```tsx
    import { PriceChange } from "@/components/ui/murphy/price-change";
    ```
    ```tsx
    <PriceChange data={[
    {
        "timestamp": 1729497600,
    ```

## Props

| Name              | Type                                      | Default |
| ----------------- | ------------------------------------------| ------- |
| `data`            | `{ timestamp: number; price: number }[]`  | `[]`    |
| `type`            | ` "%"  \| "$" `                           | `%`     |
